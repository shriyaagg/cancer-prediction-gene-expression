## Cancer Prediction Using Gene Expression Data
Classifies 5 cancer types using gene expression profiles.

**Techniques:** Mutual Information feature selection, 
One-vs-Rest Random Forest

**Dataset:** 801 samples, 8,000 genes (BRCA, KIRC, LUAD, PRAD, COAD)

**Results:** 96.67% accuracy, 98.11% F1 score
